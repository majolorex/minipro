# Lanna Culture Website

## How to Run the Website

This is a static website. You can run it using one of the following methods:

**Method 1: Open directly**
- Double-click on `index.html` to open it in your default web browser.

**Method 2: VS Code Live Server (Recommended)**
- If you have the "Live Server" extension installed in VS Code:
    1. Right-click on `index.html`.
    2. Select "Open with Live Server".

**Method 3: Python HTTP Server**
- Open a terminal in this directory and run:
  ```bash
  python -m http.server
  ```
- Then open your browser to `http://localhost:8000`.

## How to Run the Image Downloader

There is a Python script `download_images.py` to download placeholder images.

**Prerequisites:**
- Python installed.

**Run:**
```bash
python download_images.py
```
